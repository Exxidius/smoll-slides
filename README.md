# HTML Slides

A simple, minimalistic HTML-based presentation system for creating clean, lightweight slide decks.

This project is heavily inspired by and adapted from the original [minslides](https://ratfactor.com/minslides/) created by ratfactor.

## Features

*   No dependencies, just a single HTML file.
*   Navigate with keyboard keys.
*   Print-friendly styles to easily generate a PDF.
*   Easy to customize with basic HTML and CSS.

## Usage

1.  Open the `slides.html` file in any modern web browser.
2.  Use the `J` key to go to the next slide.
3.  Use the `K` key to go to the previous slide.
